## What is the GGPCL?

A free software license that allows:
- Using and modifying
- Sharing and distributing
- Viewing source code

But prohibits:
- Selling
- Commercial use
- Military use
- Illegal activities

Copyleft: All modifications must remain under GGPCL.  
Copyright: Stays with the original creator.

---

## How do I use the GGPCL?

### 1. Copy the LICENSE file to your project

### 2. Add this to the beginning of your code files:

For Shell/Bash:
```bash
#!/bin/bash
# [Your Program Name]
# Copyright (C) [Year] [Your Name]
# Licensed under GGPCL v1.0 - See LICENSE file
```

For Python:
```python
# [Your Program Name]
# Copyright (C) [Year] [Your Name]
# Licensed under GGPCL v1.0 - See LICENSE file
```

For JavaScript/TypeScript:
```javascript
// [Your Program Name]
// Copyright (C) [Year] [Your Name]
// Licensed under GGPCL v1.0 - See LICENSE file
```

For C/C++:
```c
// [Your Program Name]
// Copyright (C) [Year] [Your Name]
// Licensed under GGPCL v1.0 - See LICENSE file
```

### 3. Create a README.md for your project:

```markdown
# [Your Project Name]

[What does your program do?]

## License

Licensed under GGPCL v1.0 (Glera-Generell-Public-Creativ-License)

- Free to use and modify
- No commercial use
- No selling
- No military use

Copyright (C) [Year] [Your Name]

See LICENSE file for details.
```
